нутри content.home мы видим 2 вложенных шаблона- $image и $midle.

$image - шаблон картинки на главной. Шаблон будет использован, если в настройках главной (админка настройки/главная)   указать адрес (url) картинки. В этом варианте темы Шаблон представляет собой javascript. Надо заметить, что шаблонизатору нет никакого дела, что в шаблоне - html или javascript Выбор всегда находится в руках разработчика темы. Мы выбрали javascript по следующим причинам:
- отложенная загрузка самих картинок, несколько ускорит открытие сайта
- выбор размера картинки в зависимости от разрешения экрана, для мобильных устройств это значительно ускорит открытие сайта
- один шаблон для всех страниц, не нужно отдельно верстать страничку для главной, картинка будет только на главной
- шаблон находится внутри контентной части сайта, а требуется внести изменения в шапку сайта

Рассмотрим детально, что делает скрипт в шаблоне, для этого процитируем код:

    $("#site-description").homeimage({
      image: "$home.image",
      small: "$home.smallimage",
      breakpoint: 768,
      addclass: "home-image"
    });

Метод homeimage находится в файле /js/litepubl/effects/homeimage.min.js - поумолчанию он включено сборщиком js, поэтому, если вы хотите отключить загрузку картинок на главной, то вам следует удалить из сборщика этот файл и изменить шаблон картинки. Для работы скрипта в сборщике css подключен файл /js/litepubl/effects/css/homeimage.min.css - в случае отключения про него тоже не забудьте (равно как и при подключении этой фичи к своей теме). Рассмотрим агрументы для этого метода:

      image - адрес большой картинки , использован тег $home.image,
      small - адрес картинки для мобильных (тег $home.smallimage), не обязательный аргумент, если отсутствует или пустая строка, то будет использована одна большая картинка для всех разрешений
      breakpoint - не обязательный параметр, по умолчанию = 768, это размер окна, при котором происходит смена картинки - для десктопов одна, для мобильных устройств - другая меньшая картинка. Для темы поумолчанию на фреймворке bootstrap это 768 и это значение не надо менять. Если вы делаете свою тему, то возможно, что ширина экрана будет отличаться. для таких редких случаев как раз и оставлена возможность указать размер для скрипта.
      addclass - не обязательный параметр, по умолчанию = home-image. Это имя css класса, который будет добавлен к #site-description Определение css класса находится в файле /js/litepubl/effects/css/homeimage.min.css

$midle - используется если в настройках главной страницы (в админке Настройки/Главная) поставить галочку "Показывать дополнительные анонсы между текстом и основными анонсами" и будет выбрана рубрика в для этого. В $midle мы встречаем вложенный шаблон $post - шаблон анонса записи. Внутри шаблона content.home.midle.post можно использовать любые теги $post.*

Можно объявить этот шаблон целиком со всеми вложенными шаблонами (формат файла темы это позволяет), но тогда будет сложнее комментировать каждый из шаблонов, поэтому постараемся обойтись минимумом вложенных шаблонов.
